Was ist MediaInfo?

MediaInfo liefert technische und zusätzliche Tag-Informationen über Video- und Audiodateien.
Es ist eine freie Software (kostenlos und freier Zugriff auf den Quellcode unter GPL- oder LGPL-Lizenz)

Jetzt herunterladen!
(version 0.7.7.0, Grafische Benutzeroberfläche, mit Installationsroutine, für Microsoft Windows, 1.5 MB)
Hinweis 1 : Sie können die Versionsinformationen auf der SourceForge-Projektseite auf Neuigkeiten überwachen oder ansehen.
Hinweis 2 : Andere Versionen (Betriebssystem, Benutzeroberfläche...) sind ebenfalls verfügbar.

Sie können die Weiterentwicklung unterstützen.

ID3v1 tag, ID3v2 tag, APETag, Extended INFO tags, RIFF INFO, 16/9, 2.25

Welche Informationen liefert MediaInfo?

  • Grundsätzliches : Titel, Autor, Regisseur, Album, Titelnummer, Datum, Spieldauer...
  • Video : Codec, Bildseitenverhältnis, Bildwiederholrate, Bitrate...
  • Audio : Codec, Samplingrate, Kanäle, Sprache, Bitrate...
  • Text : Sprache der Untertitel
  • Kapitel : Anzahl der Kapitel, Kapitelübersicht

DivX, XviD, H263, H.263, H264, x264, ASP, AVC, iTunes, MPEG-1, MPEG1, MPEG-2, MPEG2, MPEG-4, MPEG4, MP4, M4A, M4V, QuickTime, RealVideo, RealAudio, RA, RM, MSMPEG4v1, MSMPEG4v2, MSMPEG4v3, VOB, DVD, WMA, VMW, ASF, 3GP, 3GPP, 3GP2

Welche (Container-)Formate unterstützt MediaInfo?

  • Video : MKV, OGM, AVI, DivX, WMV, QuickTime, Real, MPEG-1, MPEG-2, MPEG-4, DVD (VOB)...
    (Codecs : DivX, XviD, MSMPEG4, ASP, H.264, AVC...)
  • Audio : OGG, MP3, WAV, RA, AC3, DTS, AAC, M4A, AU, AIFF...
  • Kapitel : SRT, SSA, ASS, SAMI...

multiplexer, demuxer, mixer, muxer, video decoder, audio decoder, PAL, NTSC

Was kann das Programm?

  • Liest viele Video- und Audioformate
  • Verschiedene Darstellungsmöglichkeiten der Informationen (Text, Tabelle, Baumstruktur, HTML...)
  • Sie können diese Darstellungen selbst anpassen
  • Exportmöglichkeiten : Text, CSV, HTML...
  • Grafische Benutzeroberfläche, Aufruf per Kommandozeile, oder über die DLL
  • Integration in MS-Windows (Drag'n'Drop, Kontextmenü)
  • Internationalisierung : Jede Sprache wird unter jedem Betriebssystem unterstützt
  • Lokal anpassbar (Übersetzer gesucht!)
