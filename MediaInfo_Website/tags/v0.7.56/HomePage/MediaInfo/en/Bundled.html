MediaInfo is bundled with a software distribution network

The software distribution network displays advertisements during the installation of software. These advertisements allow the end user to install the advertisers' program if they want to. When user installing MediaInfo chooses to install the promoted package, revenue is generated and shared between the software distribution network and MediaInfo developers.
After the installation, there is absolutely no advertisement. No "call home" (except if you want to be informed of new releases. You can uncheck the corresponding checkbox in the preference screen, at any time, if you don't want to be informed of new versions).

Is it adware?

Some may consider the software distribution network as an adware.
On the other hand, the software distribution network does not demonstrate the negative characteristics commonly associated with adware:
  • Advertisement is only shown as one step of software installation. Once MediaInfo is installed, you do not receive any more advertisements.
  • No unwanted software is installed. User has to explicitly (Agree button) choose to install promoted software to actually receive it. You choose
  • No private information are collected. Installation program collects only information necessary to choose relevant advertisement to be displayed, such as geo-location, operating system and language.
For details see the software distribution network privacy policy.

Why did you partner with them?

Doing so allows us to generate a little bit of revenue to help continue developing, offering, and supporting our software at no cost to our users - you.

  • Use your own installer (this is open source, you can do it!)
