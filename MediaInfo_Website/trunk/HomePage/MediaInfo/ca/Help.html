Per què fer una donació?

MediaInfo és una aplicació lliure.
Llavors, per què fer una donació? Doncs per que sigui una manera de donar les gràcies, per a ajudar-nos, per a donar-nos suport... Vosaltres trieu!!!.

Donació amb targeta mitjançant SourceForge

SourceForge hospeda MediaInfo (pàgines web i les eines de desenvolupament). Fent una donació mitjançant SourceForge aportareu el 20% per a SourceForge, i el 80% al desenvolupador.
Les donacions poden ser entre 5$ i 250$
