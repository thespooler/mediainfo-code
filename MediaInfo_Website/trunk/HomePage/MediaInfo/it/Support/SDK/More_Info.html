Personalizzazione

Ottenere un oggetto specifico

MediaInfo::Get("**StreamKind**, **StreamNumber**, **Parameter**, **InfoKind**")
MediaInfo::Count_Get("**StreamKind**, **StreamNumber**")

Nota : Se **StreamNumber** è uguale a -1, si otterrà il numero di flussi di quel tipo (equivalente, ma più veloce, a MediaInfo::Get("**StreamKind**, 0, "StreamCount", Info_Text))

Nota : Solo per l'interfaccia "C" (DLL):
MediaInfo_GetI può rimpiazzare MediaInfo_Get for **Parameter** con un Integer, perché C non supporta l'operator overloading
