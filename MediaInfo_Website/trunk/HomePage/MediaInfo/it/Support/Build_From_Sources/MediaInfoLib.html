Compilare MediaInfoLib

Opzioni

Se avete intenzione di attivare il supporto solo per alcuni contenitori potete attivarlo o disattivarlo nella definizione Compilation (che si trova nel file del progetto per il vostro compilatore).
MEDIAINFO_XXX_NO o MEDIAINFO_XXX_YES (se entrambe si trovano nel file del progetto, MEDIAINFO_XXX_NO ha la priorità).
Al posto di XXX potete inserire:
  • VIDEO : tutti i video
  • AUDIO : tutti gli audio
  • RIFF : Microsoft Avi, Wav...
  • OGG : Ogg, Ogm...
  • MPEG : Mpg, Mpeg, Vob...
  • MPEG4 : Mp4, M4a, M4v...
  • MPEGA : mpa, mp2, mp3...
  • MPEGV : mpv, m2v...
  • WM : Wma, Wmv...
  • QT : Qt, Mov...
  • RM : Rm, Rmvb...
  • DVDV : Ifo of DVD
  • AAC : Aac
  • DTS : Dts
  • AC3 : Ac3
  • MK : Matroska
  • APE : Monkey Audio, Ape, Mac...
  • FLAC : Flac
  • SNDFILE : file relativi a SndFile

Compilare

Aprite il file del progetto e compilatelo.
