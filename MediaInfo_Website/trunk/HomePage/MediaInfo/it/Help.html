Non avete soldi? Che fare? non preoccuparti! ;-)

Non è un problema.
Puoi contribuire al progetto MediaInfo in diversi modi:
  • Traducete o migliorate MediaInfo nella vostra lingua d'origine.
  • Create bei settaggi HTML per MediaInfo.
  • Create belle icone per MediaInfo.
  • Traducete o migliorate il sito nella vostra lingua d'origine.
  • Inviatemi file multimediali che non sono supportati bene da MediaInfo.
  • Mandatemi i dati dei contenitori multimediali, sicché possa implementarli in MediaInfo.
  • Programmate utili funzioni (in questo caso vi prego di utilizzare l'ultima versione di CVS). E' C++
  • Supporto per le piattaforme non supportate (Linux, Win64...)
  • ...
Non esitate, contattatemi!
