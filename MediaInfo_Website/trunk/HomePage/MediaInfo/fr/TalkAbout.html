Ils parlent de MediaInfo



  • MediaInfo permet d’obtenir une multitude d’informations sur vos documents multimédias.
    L’application est en mesure d’analyser vos fichiers, puis d’indiquer notamment les codecs audio et vidéo utilisés. Un lien vers le site Web permettant de télécharger le codec est d’ailleurs disponible, vous n’installez désormais plus que les codecs dont vous avez vraiment besoin !
    MediaInfo est capable de fournir de nombreuses informations, dont l’aspect des vidéos, le nombre d’images par seconde, le débit utilisé, la fréquence d’échantillonnage des pistes audio ou encore le nombre de canaux disponibles, etc.
    Le logiciel gère plus d’une trentaine de formats parmi lesquels, MKV, OGG, Mpeg-2, Mpeg-4, IFO, DTS, AC3, MOV, RM, WMV, Aiff, IFF, APE, Flac,WMA, etc.
    MediaInfo s’intègre également au menu contextuel de l’Explorateur Windows afin d’accéder aux informations d’une séquence en quelques clics ! Enfin, l’ensemble de ces données peuvent êtres enregistrés dans un document TXT, CSV ou Html pour un usage ultérieur.

  • MediaInfo est un projet fournissant des informations étendues sur les fichiers multimédias. EN plus, le fichier MediaInfo.dll peut être copié dans le répertoire d'installation d'eMule pour permettre d'afficher encore plus d'informations sur les fichiers multimédias, informations visibles dans la boîte de dialogue Détails fichiers. Il vérifie même que l'extension du fichier correspond à l'en-tête du fichier.
