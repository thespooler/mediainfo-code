Vad är MediaInfo?

MediaInfo tillhandahåller teknisk information om video och ljudfiler.
Detta är ett gratis program (gratis att användas och gratis att använda källkoden, GPL eller LGPL licens)

Ladda ner!
(version 0.7.7.5, Användargränsnitt, med installation, för Microsoft Windows, 1.5 MB)
Notering 1: Du kan följa eller läsa vad som är nytt på SourceForge projektsida
Notering 2: andra versioner (OS, gränsnitt...) finns här

Du kan donera för att stödja en fortsatt utveckling.

ID3v1 tag, ID3v2 tag, APETag, Extended INFO tags, RIFF INFO, 16/9, 2.25

Vilken information kan jag få från MediaInfo?

  • Allmäna: titel, författare, regissör, album, spår-nummer, datum, längd...
  • Video: codec, aspect, fps, bitrate...
  • Ljud: codec, samplingshastighet, kanaler, språk, bitrate...
  • Text: språk på undertexten
  • Kapitel: antal kapitel, kapitellista

DivX, XviD, H263, H.263, H264, x264, ASP, AVC, iTunes, MPEG-1, MPEG1, MPEG-2, MPEG2, MPEG-4, MPEG4, MP4, M4A, M4V, QuickTime, RealVideo, RealAudio, RA, RM, MSMPEG4v1, MSMPEG4v2, MSMPEG4v3, VOB, DVD, WMA, VMW, ASF, 3GP, 3GPP, 3GP2

Vilka format stödjer MediaInfo?

  • Video: MKV, OGM, AVI, DivX, WMV, QuickTime, Real, MPEG-1, MPEG-2, MPEG-4, DVD (VOB)...
    (Codecs: DivX, XviD, MSMPEG4, ASP, H.264, AVC...)
  • Ljud: OGG, MP3, WAV, RA, AC3, DTS, AAC, M4A, AU, AIFF...
  • Undertexter: SRT, SSA, ASS, SAMI...

multiplexer, demuxer, mixer, muxer, video-avkodare, ljud-avkodare, PAL, NTSC

Vad kan jag göra med den här?

  • Läsa många video och ljudformat
  • Mångfald betyder sättet att visa informationen( text, blad, träd, HTML...)
  • Du kan skräddarsy dessa sätt
  • Möjlighet att exportera: text, CSV, HTML...
  • Användargränsnitt, command line eller DLL
  • Integrera med MS-Windows shell(drag'n' drop och Context meny)
  • Internationell: Visa vilket språk som helst på vilken OS version som helst
  • Lokala språk (men frivilliga behövs)
