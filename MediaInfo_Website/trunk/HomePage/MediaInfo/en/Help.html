Why donate?

MediaInfo is free software.
So, why donate? To thank, support or help. You choose.

Donation with cards via SourceForge

SourceForge hosts MediaInfo (Web pages, downloads and development tools). Donating via Sourceforge will give 20% to SourceForge, 80% to developer.
Donations can be between $5 and $250.
Donate page

No money? So what? Don't leave! ;-)

It's not shameful.
You can still contribute to the MediaInfo project in the following ways:
  • Translate MediaInfo into in your native language or improve the translation.
  • Translate this website into in your native language or improve the translation.
  • Create beautiful HTML templates for MediaInfo output.
  • Create beautiful icons for MediaInfo.
  • Send me multimedia files which are not well handled by MediaInfo.
  • Send me formats of multimedia containers, and I might be able to implement them better.
  • Program some useful functions (use the latest version of SVN please). MediaInfo is written in C++.
  • Help support currently non-supported platforms (Linux, Win64...)
  • ...
Don't hesitate to contact me!
