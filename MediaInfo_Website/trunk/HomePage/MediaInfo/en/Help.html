Why donate?

MediaInfo is free software.
So, why donate? To thank, to support, to help. You choose.

Donation with cards via SourceForge

SourceForge hosts MediaInfo (Web pages and development tools). Donating via Sourceforge will give 20% to SourceForge, 80% to developer.
Donations can be between $5 and $250
Donate page

No money? So what? Don't leave! ;-)

It's not shameful.
You can still contribute to the MediaInfo project in the following ways:
  • Translate into or improve MediaInfo program in your native language.
  • Create beautiful HTML templates for MediaInfo output.
  • Create beautiful icons for MediaInfo program.
  • Translate or improve this website in your native language.
  • Send me multimedia files which are not well handled by MediaInfo.
  • Send me formats of multimedia containers, and I will maybe be able to implement them better.
  • Program some useful functions (but use the latest version of CVS please). This is C++
  • Support currently non-supported platforms (Linux, Win64...)
  • ...
Don't hesitate to contact me!
