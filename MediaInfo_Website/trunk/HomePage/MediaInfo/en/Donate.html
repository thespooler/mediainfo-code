Donate to MediaInfo

MediaInfo is a free software, so why donate?

In order for MediaInfo to be in a full-time development state, your support and help is greatly appreciated. Since MediaInfo is a free software, your donation is the best way to encourage the author and make it possible to continue moving forward on the project.

No money to donate? Don't leave! ;-)

You can still contribute to the MediaInfo project in many other ways such as:
  • Translate MediaInfo into your native language or make improvements to the existing translations,
  • Translate this website into your native language or make improvements to the existing translations,
  • Create beautiful HTML templates for MediaInfo's output,
  • Create beautiful icons for MediaInfo,
  • Send me multimedia files which are not well handled by MediaInfo,
  • Send me formats of multimedia containers, and I might be able to implement them better,
  • Program some useful functions (use the latest version of SVN please). MediaInfo is written in C++,
  • Help support currently non-supported platforms (Linux, Win64...),
  • ...
Don't hesitate to contact me if you're interested in contributing in any way!
